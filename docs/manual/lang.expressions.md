# Expressions
An expression in ShortHand is any language construct that has a _value_ of any type and can be used whenever the value is required.  
  
ShortHand has the following types of expressions:   
  
1. | [Literal Numbers](lang.number.md)  
---|---  
2. | [Literal Strings](lang.string.md)  
3. | [NULL Value](lang.null.md)  
4. | [Variables](lang.variables.md)  
5. | [Arithmetic Expresssions](lang.arithmetic.md)  
6. | [String Expressions](lang.string.md)  
7. | [Boolean Expressions](lang.boolean.md)  
8. | [Function Calls](lang.call.md)  
9. | [Object Constructors](lang.objects.constructor.md)  
10. | [Numbered Arrays](lang.arrays.md)  
11. | [Hash Arrays](lang.hashes.md)
