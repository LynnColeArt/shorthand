# Statements
ShortHand program consists of _statements_ \(also called _operators_\). Each statement performs some action when it is executed by ShortHand engine - assigns value to a variable, prints value, calls function etc. Some complex statements \(like loops\) may include other statements.  
  
Statements are normally separated from each other using by newlines or blocks of unescaped HTML code \(see [Statements Separation](lang.separation.md)\). Some compound statements, like if-then-else, span multiple lines.  
  
Below is a complete list of statements that exist in ShortHand.   
  
|   
---|---  
Operator | Action  
[Assignment](lang.assignment.md) | Assigns arbitrary expression to a variable.  
[Function Invocation](lang.call.md) | Invokes built-in or user-defined function.  
Method Invocation | Invokes method of an object variable.  
[PRINT](lang.print.md) | Prints expression to standard output stream.  
[IF-THEN-ELSE](lang.if.md) | Conditionally executes a blocks of statements.  
[FOR Loop](lang.for.md) | Repeatedly executes a block of statements by iterating a range of numbers.  
[WHILE Loop](lang.while.md) | Repeatedly executes a block of statements while condition is satisfied.  
[GRID Loop](lang.grid.md) | Executes a block of statement for every records in SQL recordset.  
[BREAK](lang.break.md) | Breaks execution of a loop.  
[CONTINUE](lang.continue.md) | Restarts execution of a loop from next iteration.  
[INCLUDE](lang.include.md) | Includes and executes another ShortHand file.  
[JUMP](lang.jump.md) | Transfers control to another ShortHand file.  
[DEFINE FUNCTION](lang.defun.md) | Defines new function.  
[RETURN](lang.return.md) | Returns a value from user-defined function.  
[LOCAL](lang.local.md) Declaration | Declares local variable within user-defined function
