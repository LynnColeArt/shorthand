# Types
Shorthand operates with the following data types:  

  * [Numbers](lang.number.md)
  * [Strings](lang.string.md)
  * [Dates](lang.dates.md)
  * [Objects](lang.objects.md)
  * [Numbered Arrays](lang.arrays.md)
  * [Hash Arrays](lang.hashes.md)
  * [NULL](lang.null.md)

These are the types thate are internally used to store ShortHand values. In most cases you don't have to worry about data types. ShortHand engine automatically determines types of expressions and supplies type conversions when necessary.   
  
Until some value is assigned to a variable, the variable has NULL value, which does not belong to any type, but can be though of as an empty string \(and it behaves as empty string\).  
  
[Expressions](lang.expressions.md) section contains more information about behaviour of different types.
