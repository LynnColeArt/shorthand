# What's new in Shorthand 1.2 ???
**Enhanced Database Options:**  
As of version 1.2, Shorthand now supports [ODBC](builtin.ODBC.md) \(open database connectivity\) on Windows and Unix.
**ISAPI NOW AVAILABLE:**  
1.2 now totes an enhanced version for Microsoft Internet information server. The ISAPI implementation of the Shorthand scripting languages boasts faster speeds, more economic use of server resources, and improved thread management.
**Extending the optional data type:**  
As a rule, Shorthand determines the data type it is using dynamically. 
Shorthand has had user defined data typing for dates since version 0.9.6. But as of version 1.2.0 optional data types [string](builtin.string.md), [int](builtin.int.md), and [float](builtin.float.md), have been added to the Shorthand engine and language. For ease of use they are addressed as functions.
**Bug Fixes:**  
Shorthand 1.2 fixes all of the known bugs from previous versions of the Shorthand engine including the [Replace\(\)](builtin.replace.md) large string bug.
