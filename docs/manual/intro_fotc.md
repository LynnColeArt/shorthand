#### **Introductory Tutorial**
Shorthand is a high level scripting language designed specifically for web based applications. In this tutorial we will cover the two most important things in any programming language. 
_**They are as follows:**_  
  
**1.** The Print statement  
**2.** Variables  
Let’s get started\!
The print statement is used to output text or variables to a web page.
**Example:**
`<~  
Print "Hello World"  
~>` The Print statement can also be used by creating a variable and printing it. **Example:** | <~  
` MyVariable = "Hello World"  
  
Print MyVariable  
~>`  
---  
Speaking of variables... you can also use the following syntax:
`<~  
MyVariable = "Hello World"  
~>   
  
<~@MyVariable~> `  
---  
The **@** symbol indicates that you want to print a specific variable. But it only works at the beginning of a line.
With any of the above scripting techniques the output will read “Hello World.”
And there you have it… you have just written your first Shorthand script.
Good Luck\!
