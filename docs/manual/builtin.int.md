# INT
Converts string to integer.  

#### Syntax
**int**\(number\)  

#### Parameters
_number_
    Any numeric value
#### Return Value
    Returns a numeric value. Generally used for mathematical equations that require more than one kind of data type. 
#### Remarks
    none 
#### Example
    
    
    f = 26
    g = Int(f)
    
#### Compatibility
    ShortHand **1.1.4b** or higher
