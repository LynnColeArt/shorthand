# Statements Separation
Statements in ShortHand within tags \(see [Basic Syntax](lang.basic-syntax.md)\) are separated by new-line characters - i.e. only one statement per line is allowed.  
  

    
    <~
      print "Hello"
      print " World"  
    ~>
      
---  
  
When you use multiple tags, there is no need to put extra new lines between statements:  
  

    
    <~ print "Hello" ~> <~ print " World" ~>   
---
