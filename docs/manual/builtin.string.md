# String
Converts input to string.  

#### Syntax
**String**\(input\)  

#### Parameters
_input_
    Any Value  

Return Value
    Returns a string. Generally used for mathematical equations that require more than one kind of data type. 
#### Remarks
    none 
#### Example
    
    
    f = date(now())
    g = String(f)
    
#### Compatibility
    ShortHand **1.1.4b** or higher
