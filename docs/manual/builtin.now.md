# now
Returns current date and time  

#### Syntax
**now**\(\)
#### Parameters
_none_
#### Return Value
    new [date](lang.dates.md) value that corresponds to current local date and time
#### Remarks
    function `now()` derives date/time value from the computer system clock. 
#### Example
    
    
        d = AddHours(now(), 12)
    
#### Compatibility
    ShortHand **1.0** or higher
#### See Also
    [Dates](lib.dates.md), [AddHours](builtin.addhours.md), [AddMinutes](builtin.addminutes.md), [AddSeconds](builtin.addseconds.md)
