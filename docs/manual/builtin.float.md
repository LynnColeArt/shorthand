# FLOAT
Converts string to Decimal.  

#### Syntax
**int**\(number,spaces\)  

#### Parameters
_number_
    Any numeric value  

_Spaces_
    The number of spaces from the right in which to place the decimal point.  

#### Return Value
    Returns a decimal point number. Generally used for mathematical equations that require more than one kind of data type. 
#### Remarks
    none 
#### Example
    
    
    f = 26
    g = Float(f,2)
    
#### Compatibility
    ShortHand **1.1.4b** or higher
