# Introduction to Shorthand
In response to challenges faced by the web development community, the Shorthand Scripting Language was designed as an easier way to design and deploy cross platform web applications. The result is a robust scripting language that is easy to learn, install, and work with.  
ShortHand engine can be added to any Web server. Present version of ShortHand also provides convenient built-in MySQL and ODBC integration. 
This help file is designed to help you get your bearings, and get started programming in the Shorthand scripting language.
Enjoy\!
