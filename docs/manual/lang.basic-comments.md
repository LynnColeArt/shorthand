# Comments
ShortHand has only one-line comments. A comment starts with a single quote character and lasts until the end of current line or the end of ShortHand tag \(~>\), whatever comes first. 
    
    <~
       i = 1 ' This is comment
    ~>
    
    <~ s = 'Hello' ' This is another comment ~>
      
---  
Note that single-quote character can also be used to specify string literals. ShortHand automatically distingushes between single-quoted strings and comments.
